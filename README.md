## 게임 설명
유니티짱이 지옥요새 안에서 탈출해서 지상으로 올라가는 것을 목표로 하는 게임입니다.

### 조작키
스페이스 바 : 점프 (holding)
방향키 좌,우 : 이동
p 또는 ESC : 일시정지

### 장애물
빨간색 장판 : 해당 방향으로 튕겨냅니다.
초록색 장판 : 화살표 방향(장판 안에 작게 보입니다.)으로 튕겨냅니다. (좌 또는 우)
대포 : 일정 시간마다 총알을 발사합니다.
벽 : 이동을 방해하는 장애물 입니다.

### 목표
스테이지 5의 끝부분에 도달하여 지옥요새를 탈출하세요!

## 알려진 문제
- 게임 첫 화면에 튜토리얼창이 띄워진 상태에서 창 사이즈를 변경하면 하이라이트가 그에 맞춰 재배치 되지 않는 문제가 존재합니다.
- 게임을 일시중지 할 떄 캐릭터가 바닥아래를 뚫고 떨어질 수 있는 문제가 존재합니다.
- 게임이 시작되지 않았음에도 음악이 재생되는 문제가 존재합니다.
